# Machine Learning based gene feature selection

## An interactive platform for **gene feature selection** from high-dimensional omics data.

## It integrates multiple widely used feature selection methods into a **single, user-friendly dashboard**.


## Key Features

``` bash

- Upload significant gene lists, expression matrices, and metadata

- Multiple feature selection methods:

  1. **LASSO (L1-regularized Logistic Regression)**

  2. **mRMR (Minimum Redundancy Maximum Relevance)**

  3. **Random Forest feature importance**

- User-defined **Top-N gene selection**

- Flexible gene ID column selection (ENSEMBL, SYMBOL, etc.)

- Interactive tables with CSV export


```
